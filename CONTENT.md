# CONTENT.md

## Website Goal

Create a modern portfolio website for a senior backend/full-stack engineer who builds:

- custom web applications
- MVPs
- internal tools
- business automation systems
- AI-assisted workflows

The website should attract:
- startups
- founders
- small businesses
- product teams
- recruiters for product/backend roles

The site should feel:
- modern
- practical
- trustworthy
- product-oriented
- technically strong
- easy to read

Not:
- corporate enterprise
- generic freelancer
- “I make websites”
- overly technical

The main impression should be:

“This person can actually build real products and business systems.”

The writing should sound natural.
